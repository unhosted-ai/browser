tracker-blocker

← back to delta

a browser that ships with privacy on by default. nothing to install, nothing to configure, nothing reported back.

delta hooks electron's webRequest api at startup and inspects every outbound request against a static list of about 150 known third-party trackers — analytics, ad networks, fingerprinters, session replayers. requests to those hosts are short-circuited with an abort response; the page never sees them, and the network never sees you.

which trackers

two sets, layered. the curated short list (~200 entries, hand-vetted) at tracker-list.ts carries rich metadata — each entry has an owner label (google, meta, amazon, hotjar) so the privacy report can roll up counts by company, not just by domain.

on top of that, an extended list from EasyPrivacy — ~42,000 known trackers as of the last refresh — is bundled in tracker-list.easyprivacy.ts. it's regenerated by a small script (apps/browser/scripts/build-tracker-list.mjs) and committed so users don't need network at install time. settings → extended tracker list flips it off if it ever blocks something it shouldn't (the curated short list keeps working). EasyPrivacy is dual-licensed GPL-3.0 / CC-BY-SA-3.0; attribution sits at the top of the generated file.

the 30-day report

blocked requests increment a daily counter in memory and on disk. settings → privacy renders a small report:

  • total trackers blocked across all your tabs
  • which sites had the most
  • which company those trackers belong to
  • a 30-day spark line of daily counts

the report never leaves your machine. it isn't synced, isn't uploaded, isn't shared with us — there's no "us" to share it with. privacy.reset() wipes the counters; the blocking still runs from the static list.

per-tab chip

the active tab's address bar shows a small shield with the count of trackers blocked since the last top-level navigation. tap it to jump to the privacy section in settings. the chip on the new tab summarises the trailing 30 days.

not a full ad blocker

this is intentionally narrow: trackers, not all advertising. blocking the entire ads-and-creator-economy stack is a different project with different trade-offs. delta blocks the surveillance layer; if you want full ad-blocking too, plug in ublock origin or a system-wide solution like nextdns.

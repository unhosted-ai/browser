no telemetry. no account.

device is the identity. conversations, bookmarks, history live as flat json on your disk. quit deletes nothing; rm -rf does.

three commands.
no account.

install ollama, pull a model, clone delta. the assistant auto-detects the local llm on first launch.

# 1. install ollama (mac/linux/windows)
brew install ollama

# 2. pull a small model + start the server
ollama pull llama3.2
ollama serve

# 3. clone + run delta — auto-detects ollama
git clone https://github.com/Delta-Practice/Browser
cd Browser && pnpm install && pnpm delta:dev
